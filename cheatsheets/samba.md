#Samba
Below you can find the cheats for a samba server RHEL7.
Also a checklist for troubleshooting the server.
##Cheats
###Installation
* ´libsemanage-python´
* ´samba-common´
* ´samba´
* ´samba-client´
###Firewall
* ´firewall-cmd [--zone=<zone>] --add-service=samba [--timeout=<seconds>]´ samba service add to firewall
